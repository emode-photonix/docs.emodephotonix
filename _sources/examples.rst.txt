
Examples
========

To launch EMode, initialize a simulation, and check out an active license:

.. tabs::
    
    .. code-tab:: python
        :caption: Python
        
        import emodeconnection as emc
        em = emc.EMode()
    
    .. code-tab:: matlab
        :caption: MATLAB
        
        em = emodeconnection();

Next, EMode functions are called with:

.. tabs::
    
    .. code-tab:: python
        :caption: Python
        
        em.function_name(keyword_1 = value_1, keyword_2 = value_2, ... )
    
    .. code-tab:: matlab
        :caption: MATLAB
        
        em.function_name(keyword_1 = value_1, keyword_2 = value_2, ... );

Close EMode and check in the license with:

.. tabs::
    
    .. code-tab:: python
        :caption: Python
        
        em.close()
    
    .. code-tab:: matlab
        :caption: MATLAB
        
        em.close();
